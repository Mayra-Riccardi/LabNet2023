Práctica Angular

Descripción del Desafío

Consumir la API desarrollada en la entrega número 7 del lab, el desafío de Web Api.
Agregarle estilos, formularios reactivos, validaciones y estructura al proyecto y librerías.
Incluir formularios reactivos y mejoras en la interfaz de usuario.

Durante el Lab aprendimos a manejarnos con Visual Studio, Visual Studio Code, utilizamos .NET Framework, C#, Js y Angular

Fueron 8 prácticas, empezando por Programación orientada a objetos. Manejo de excepciones, SQL, estructura del proyecto y validaciones.
Luego pasamos al desafío de MVC, para continuar con el de Web Api y finalizar con la integración de los dos mundos, Brackend y Front End

Tecnologías Utilizadas

- .NET Framework en C#

- Angular

- Angular Material

- Bootstrap

- SweetAlert

- GitHub
